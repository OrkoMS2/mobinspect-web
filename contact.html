How to Report Issues Effectively

üêõ Bug Reports

When reporting a bug, please include:

  • Detailed description of the issue
  • Steps to reproduce the problem
  • Expected vs actual behavior
  • Device model and Android version
  • MobInspect app version
  • Screenshots or screen recordings (if applicable)

üîç False Positive Reports

If you believe an app was incorrectly flagged:

  • Name of the flagged app
  • Source where you downloaded it
  • Why you believe it's safe
  • Screenshot of the scan results
  • App package name (if known)

üí° Feature Requests

For feature suggestions, tell us:

  • What feature you'd like to see
  • Why it would be useful
  • How you envision it working
  • Any similar features in other apps
  • Priority level for you
